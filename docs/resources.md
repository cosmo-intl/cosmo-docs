# Resources

External references for working with ICU, CLDR and locale data — the same data
every Cosmo port draws on.

- [ICU Documentation](https://unicode-org.github.io/icu/) — the user guide for International Components for Unicode.
- [ICU Data](https://github.com/unicode-org/icu/tree/release-65-1/icu4c/source/data) — the raw CLDR-derived data tables ICU ships.
- [Online ICU Message Editor](https://format-message.github.io/icu-message-format-for-translators/) — author and preview ICU MessageFormat strings.
- [ICU data tables](https://intl.rmcreative.ru/) — browsable locale data, by Alexander Makarov.
- [The Locale Explorer](http://www.flounder.com/localeexplorer.htm) — locale data viewer, by Joseph M. Newcomer.
